# Acme Online Payments API
Make payments and set up recurring billing for Acme Online.

## Usage
Documentation for usage pending.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'payments'
```

And then execute:
```bash
$ bundle
```
